Calculus II — §6.1 Area Between Curves
← Back to Calc II graph  

This is a source of §5.2- Definite Integrals. Recall that or original understanding, in §5.1 Areas & Distances, of an integral was the area underneath a curve. These can be treated like shapes, allowing us to "add" and "subtract" them from one another. This motivates the following:

[!theorem] Proposition The area between the curves $y = f(x)$ and $y =g(x)$ between $x = a$ and $x = b$ is given by: $$ > \int_a^b \left | f(x) - g(x) \right |dx > $$
Problems like these often involve integrating an absolute value symbol, which can be tricky! Remember to sort the interval $[a,b]$ into the regions where $f(x) \geq g(x)$ and $f(x) \leq g(x)$.
[!example] Example Compute the area between the curves: $$ > f(x) = 4x - x^2 \quad \text{and} \quad g(x) = x^2 > $$
> >> [!success]- Solution >> From the image, we know that the correct integral to set up is: >> $$\int_0^2 f(x) - g(x) dx = 2 \int_0^2 2x - x^2 dx = 2 \left [ x^2 - \frac{x^3}{3} \right ]_0^2 = 2 \left ( 4 - \frac83 \right ) = \frac83$$
[!example] Problem Compute the area between the curves: $$ > f(x) = \frac{8}{x}, \quad \text{ } \quad g(x) = 2x, \quad \text{ and } \quad x = 4. > $$
>> [!success]- Solution >> First we graph >> >> and integrate: >> $$ \int_2^4 2x - \frac8x dx = \left [ x^2 - 8\ln(x) \right ]_2^4 $$
[!example] Problem Sketch the region enclosed by the following curves, and then set up an integral to compute the area of these regions: $$y=x \sqrt{x^2+1}, y=e^{-\frac{1}{2} x}, x=-3 \quad \text{and the} \quad y\text{-axis.}$$
>> [!success]- Solution >> Show students here that it’s not necessary to know the exact shape of a graph so long as they can find intersection points and know the region of integration. >> The correct integral would be: $$\int_{-3}^0 e^{- \frac12 x} - x \sqrt{x^2+1} dx$$
