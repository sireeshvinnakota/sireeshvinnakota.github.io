Calculus II — §8.1 Arc Length

This is another source of §5.2- Definite Integrals

If $f’$ is a continuous function on the interval $[a,b]$, then the length of the curve $y = f(x)$ between $a \leq x \leq b$ is:
$$ L := \int_a^b \sqrt{1 + \left [ f’(x) \right ]^2} dx $$

[!example] Example Compute the length of the curve $y^2 = x^3$ between $(1,1)$ and $(4,8)$.
>>[!success]- Solution >>Since both of these coordinates are in Quadrant 1, our square root is well defined: $$y = x^\frac{3}{2}$$ We can proceed just following the formula: $$ L = \int_1^4 \sqrt{1 + \left [ \frac32 x^{1/2} \right ]^2}$$
