Calculus II — §5.5 The Substitution Rule
← Back to Calc II graph  

This is the first of many techniques that we will learn to find functions $F$, corresponding to a given function $f$, as in [[§5.3 The Fundamental Theorem of Calculus#^3e560b|the FTC]].

[!theorem] Proposition If $g’$ is a continuous on $[a,b]$ and $f$ is continuous on the range of $u = g(x)$, then: $$ > \int_a^b f(g(x))g’(x)dx = \int_{g(a)}^{g(b)} f(u)du > $$ For indefinite integrals, we have: $$ > \int f(g(x))g’(x)dx = \int f(u)du > $$
For functions that are even or odd, we have little shortcuts that can help us evaluate integrals quickly: $$ \begin{array} {|c|c|c|} \hline \text{Type} & \text{Definition} & \text{Integral Property} \\ \hline \text{$f(x)$ is Even} & f(-x) = f(x) & \displaystyle\int_{-a}^a f(x) dx = 2\int_0^a f(x)dx \\ \text{$f(x)$ is Odd} & f(-x) = -f(x) & \displaystyle \int_{-a}^a f(x) dx = 0 \\ \hline \end{array} $$ Use this as a trick to potentially bypass tricky problems!
[!example] Example Compute the definite integral: $$ > \int_1^2 \frac{dx}{(3-5x)^2} > $$
>> [!success]- Solution >> Let $u = 3 - 5x$, for which the problem reduces to [[§5.4 Indefinite Integrals#^5fcf9e]].
[!example] Problem Compute the following definite integral: $$ > \int_e^{e^4} \frac{1}{x \sqrt{\ln(x)}} > $$
>> [!success]- Solution >> Let $u = \ln(x)$, then [[§5.4 Indefinite Integrals#^5fcf9e]]
[!example] Problem Compute the following indefinite integrals
  • $\int \sin^2 \theta \cos \theta d\theta$
  • $\int \sec^2 2\theta d \theta$
  • $\int \frac{\cos (\ln(t))}{t} dt$
  • $\int \tan^2 \theta \sec^2 \theta d\theta$
>>> [!success]- Solution >> We’ll list the $u$ choices (I’ll complete one to make sure we know what we’re doing) >> - $u = \sin(x), du = \cos(x)dx$ We can directly make a substitution: >> $$\int \sin^2 \theta \cos(\theta) d\theta = \int u^2 du = \frac{u^3}{3} + C$$ >> This is an answer in terms of $u$, so to complete the problem, we must substitute back: >> $$\frac{u^3}{3} + C = \frac13 \sin^3 (x) + C$$ >> completing our solution. >> - $u = 2\theta$ >> - $u = \ln(t)$ >> - $u = \tan(\theta)$ >> Refer to the [[§5.4 Indefinite Integrals#^adaaae|trig table]] when needed
[!example] Problem (Challenge) Compute the following integral: $$\int_0^1 \frac{dx}{(1 + \sqrt{x})^4}$$
>> [!success]- Solution >> Final answer is $\frac16$. Here’s a solution: Let $u = \sqrt{x} + 1$ and notice that $dx = 2(u-1)$ (rearrange the differentiated equation). This gives you the integral: >> $$\int_1^2 \frac{u-1}{u^4}du$$which can be split into two integrals, each of which are computable to us.
[!example] Problem (Challenge) Compute the following indefinite integrals.
  • $\displaystyle \int_{-\pi/2}^{\pi/2} \frac{x^2 \sin(x)}{x^6 + 1} dx$
  • $\displaystyle \int \sqrt{x-1}(x+3) dx$
  • $\displaystyle \int \sqrt{x} \sin(\sqrt{x^3} +1)dx$
>>> [!success]- Solution >> The first is 0 since it is an odd function. Second, substitute $u = x -1$; then $x+ 3 = u + 4$ and $du = dx$. Finally, the last one is $u = \sqrt{x^3} + 1$ and $du = \frac32 \sqrt{x}$
[!example] Problem (Challenge) Suppose $f$ is a function with $\displaystyle \int_0^8 f(x)dx = 20$. What is the value of $\displaystyle \int_0^2 f(4x)dx$?
>> [!success]- Solution >> Using $u = 4x$, we see that: >> $$\int_0^2 f(4x) dx = \frac14 \int_0^8 f(u) du = \frac14 \cdot 20 = 5$$
