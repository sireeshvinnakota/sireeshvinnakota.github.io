Calculus II — §6.2 Volumes of Rotation
← Back to Calc II graph  

This is a source of §5.2- Definite Integrals

Volumes by washers or disks are an important application of integration, and can be computed using the formulas:
$$ \underbrace{V = \pi \int_a^b \left [ R(x) \right ]^2 dx}_{\text{Disk}} \quad \text{and} \quad \underbrace{V = \pi \int_a^b \left [ R(x)^2 - r(x)^2 \right ] dx}_{\text{Washer}} $$
where $R(x)$ represents the “radius” or distance between the function and the axis of rotation, and, in the case of the Washer method, $r(x)$ represents the “inner radius.”

[!example] Example Find the volume of the solid obtained by rotating the region bounded by the given curves about the specified line. Sketch the region, and state the method of integration before computing the integral: $$ > y = 1-x^2 \quad \text{ about the } \quad x-\text{axis} > $$
>> [!success]- Solution >> [Graph](https://www.geogebra.org/m/zBRtUVfR). This is the disk method, so we have the integral: >> $$ V = \pi \int_{-1}^1 \left ( 1-x^2 \right )^2 dx$$
[!example] Problem Find the volume of the solid obtained by rotating the region bounded by the given curves about the specified line. Sketch the region, and state the method of integration before computing the integral: $$ > y = 6 - x^2, \quad y = 2 \quad \text{ about the } \quad x\text{-axis} > $$
>> [!success]- Solution >> The correct integral to set up is:
$$V = \pi \int_{-2}^2 (6-x^2)^2 - (2)^2 dx$$
>> using the Washer method. From here, integration is straightforward.
[!example] Problem Find the volume of the solid obtained by rotating the region bounded by the given curves about the specified line. Sketch the region, and state the method of integration before computing the integral:
  • $y = x^3, \quad y = 0, \quad x = 1 \quad \text{ about the } \quad x = 2$
  • $x = y^2, \quad x = 1-y^2 \quad \text{ about the } \quad x=3$
>>> [!success]- Solution >> Both need one to convert equations to functions of $y$. >> - Using $x = y^{1/3}$ we see:$$V = \pi \int_0^1 (y^{1/3})^2 - 1^2 dy$$ >> - Our bounds of integration here are $y = \pm \sqrt{2}/2$ (we find these in the same way we found them in [[§6.1 Area Between Curves#^0bf336|§6.2]]): >> $$V = \pi \int_{-\sqrt{2}/2}^{\sqrt{2}/2} (y^2)^2 - (1-y^2)^2 dy$$
