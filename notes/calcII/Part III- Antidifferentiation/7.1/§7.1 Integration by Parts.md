This section is just the "opposite" of the product rule. 
> [!theorem] Proposition
> [Integration by Parts]
> We have the following integration identity:
> $$
> \int u dv = uv - \int v du
> $$

Hereâ€™s a cool rule of thumb that I found to always select the correct $u$ and $v$: **Choose $u$ in this order:**
- L: logarithms
- I: inverse functions
- A: algebraic (polynomials, etc.)
- T: trigonometric functions
- E: exponentials
I found out later that if we add a $d$ at the end, this spells DETAIL backwards... so another way to remember this trick is:
- D: $dv$
- E: exponentials
- T: trigonometric functions
- A: algebraic (polynomials, etc.)
- I: inverse functions
- L: logarithms


> [!example] Problem
> Compute the antiderivative:
> - $\displaystyle \int (x^2 + 2x)\cos(x) dx$
> - $\displaystyle \int 4x \cos (2-3x) dx$
>
>> [!success]- Solution
>> - We show the following steps:
>> $$\begin{alignat*}{2}\int (x^2 + 2x)\cos(x) dx &= \int (x^2 + 2x) d \sin(x) \\&= \left ( x^2 + 2x \right ) \sin(x) - \int \sin(x) d x^2 + 2x\\&= \left ( x^2 + 2x \right ) \sin(x) - 2 \int \sin(x) (x+1) dx\\&= \left ( x^2 + 2x \right ) \sin(x) - 2 \int (x+1) d \left ( -\cos(x) \right )\\&= \left ( x^2 + 2x \right ) \sin(x) + 2 \left ( (x+1) \cos(x) - \int \cos(x)dx \right )\\&= \left ( x^2 + 2x \right ) \sin(x) + 2 \left ( (x+1) \cos(x) \right ) -\sin(x) + C\end{alignat*}$$
>> - Start with a change of coordinates:
>> $$u = 2-3x \quad \text{ , } \quad du = -3dx \quad \text{ and } \quad x = \frac{u - 2}{-3}$$From here, we can rewrite:$$\int 4x \cos(2-3x) dx = \frac{-4}{3} \int \underbrace{x}_{= \frac{u - 2}{-3}} \cos (\underbrace{2 - 3x}_{u}) \underbrace{-3x dx}_{du} = \frac{-4}{3} \cdot \frac{1}{-3} \int (u-2) \cos (u) du$$which can be solved the same way as the preceding problem.


> [!example] Problem
> Compute the antiderivative:
> - $\displaystyle \int e^x \sin (x) dx$
> - $\displaystyle\int \ln\sqrt{x} dx$
> - $\displaystyle \int e^{\sqrt{x}} dx$
> - $\displaystyle \int \cos(\ln(x))dx$
> - $\displaystyle \int (\arcsin(x))^2 dx$
>
>> [!success]- Solution
>> We can use tabular integration to speed these up.
>> - Consider the following table:
>> $$\begin{array}{|c|c|}\hline \frac{d}{dx} & \int dx \\ \hline\sin(x) & e^x \\ \cos(x) & e^x\\ \sin(x) & e^x\\\cos (x) & e^x \\ \hline\end{array}$$
>> This gives us: $$ \begin{alignat*}{2}\int e^x \sin(x)dx &= e^x \sin(x) - \int e^x d(\sin (x)) \\ &= e^x \sin(x) - \int e^x \cos(x)dx\\ &= e^x \sin(x) - \left [ e^x \cos(x) - \int e^x d(\cos(x)) \right ]\\ \underbrace{\int e^x \sin(x)dx } &= e^x \sin(x) - e^x \cos(x) - \underbrace{\int e^x \sin(x) dx}\end{alignat*}$$
>>Looking at the last line, we see that:
>> $$\int e^x \sin(x) dx = \frac12 e^x (\sin(x) - \cos (x))$$
>> - Using $u$-substitution, we see that:
>> $$u = \sqrt{x} \quad \text{ and } \quad du = \frac{1}{2 \sqrt{x}} dx \quad \text{ give us } \quad dx = 2 u du$$
>> This converts our integral to
>> $$\int \ln(\sqrt{x}) = 2 \int u \ln (u) du$$
>> which we can compute using the dETAIL technique.
>> - Same technique as above gives us:
>> $$ \int e^{\sqrt{x}} dx = 2 \int ue^u du$$
>> - Making a $u$-substitution for $\ln(x)$ gives you:
>> $$\underbrace{u = \ln(x)}_{e^u = x} \quad \text{ and } \quad du = \frac1x dx \quad \text{ implying } \quad dx = e^u du$$
>> This turns the integral into:
>> $$\int\cos (\ln(x))dx = \int e^u \cos(u) du$$
>> which we computed in the example above. 
>> - This is another $u$-[[Â§5.5 The Substitution Rule|substitution]], this time: $$\underbrace{u = \arcsin (x)}_{x = \sin(u)} \quad \text{ and } \quad du = \frac{1}{\sqrt{1-x^2}} dx$$
>> Solving for $u$ in the second equation, we see:
>> $$ dx = \sqrt{1 - (\sin(u)^2)} du = \cos(u) du$$
>> by the trigonometric identity $\sin^2(x) + \cos^2(x) = 1$. This gives us the integral:
>> $$\int \arcsin^2 (x) dx = \int u^2 \cos(u) du$$
> for which a similar integral was computed earlier.
