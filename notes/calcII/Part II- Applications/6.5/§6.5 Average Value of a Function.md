> [!theorem] Proposition
> Let $f$ be an integrable function defined on the interval $[a,b]$. Then the average value of $f$, $f_{avg}$ on the defined interval is:
> $$
> f_{avg} := \frac1{b-a} \int_a^b f(x) dx
> $$

> [!theorem] Theorem (Mean Value Theorem for Integrals)
> If $f$ is a continuous function on the interval $[a,b]$, then there exists $c \in [a,b]$ for which 
> $$
> f(c) = f_{avg} \quad \text{ that is } \quad f(c) (b-a) = \int_a^b f(x)dx
> $$

The geometric interpretation of this equation is finding the real number $c$ such that the area cut out by the function is split directly in half by $f(c)$, slicing horizontally.

> [!example] Example
> Find the average value of the following function on the given interval:
> $$
> f(x) = \cos^4(x) \sin (x)  \quad \text{ defined on } \quad [0,\pi]
> $$
>> [!success]- Solution
>> Using the theorem above, we can compute:
>> $$f_{avg} = \frac{1}{\pi - 0} \int_0^\pi \cos^4 x \sin(x) dx$$
>> We may proceed by $u$-[[§5.5 The Substitution Rule|substitution]], on $u = \cos (x)$ to find the answer.

> [!example] Problem
> Compute the average value of the function:
> $$
> f(x) = 8x - 3 + 4e^{2-x} dx \quad \text{ on the interval } \quad [0,2]
> $$
>> [!success]- Solution
>> We set up the same integral as before:
>> $$ f_{avg} = \frac12\int_{0}^2 8x - 3 + 4e^{2-x}dx$$
>> The only twist in the remainder of the problem is the antiderivative of $e^{2-x}$, which is just $-e^{2-x}$. Use [[§5.4 Indefinite Integrals#^5fcf9e|power rule]]

> [!example] Problem
> The temperature (in $F$) $t$ hours after 9 a.m. is given by the function:
> $$
> T(t) := 50 + 14 \sin \left ( \frac{\pi t}{12} \right )
> $$
> Compute the average temperature between 9 a.m. and 9 p.m. using the function above.
>> [!success]- Solution
>> The integral to set up is:
>> $$T_{avg} = \frac{1}{12} \int_0^{12} T(t) dt$$
