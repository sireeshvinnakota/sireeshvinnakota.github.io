The main idea of this section is that we can try to approximate areas of non-standard regions of the plane using rectangles, and then adding more rectangles to decrease error. First, we’ll review sigma notation:

>[!note] Note (Sigma Notation)
>Sums can be concisely expressed using notation in which one utilizes an *index*, or variable that takes on consecutive number values. For example: $$\sum_{i = 4}^7 2i = 2(4) + 2(5) + 2(6) + 2(7) = 8 + 10 + 12 + 14 = 44$$ Recall that distributivity (pulling out constants) and associativity (splitting sums into two sums) both work. Also: $$\sum_{j= 1}^n j = \frac{n (n+1)}{2} \quad \text{,} \quad \sum_{j = 1} ^ {n} j^2 = \frac{n (n+1)(2n+1)}{6} \quad \text{ and } \quad \sum_{j= 1}^n j^3 = \left ( \frac{n (n+1)}{2} \right )^2$$

> [!definition] Definition (Riemann Sum)
> Let $f(x)$ be a continuous function. we *approximate* the area between $f(x)$ and the $x$-axis on the interval $[a,b]$ using $n$ rectangles via:
> $$ \sum_{i = 1}^n f(x_i^*) \Delta x $$
> where:
> $$ x_i^* = \begin{cases} a + (i-1) \Delta x & \text{ for Left-hand Sum, written } L_n\\ a + i \Delta x & \text{ for Right-hand Sum, written } R_n\\ a + \frac{i}{2} \Delta x & \text{ for Midpoint Sum, written } M_n\\ \end{cases} \quad \text{ and } \quad \Delta x = \frac{b - a}{n} $$

> [!definition] Definition
> To compute the *exact area* of this region between $f(x)$ and the $x$-axis on the interval $[a,b]$, we use:
> $$ A = \lim_{n \to \infty} L_n =  \lim_{n \to \infty} R_n $$

> [!example] Example
> Expand the following sums:
> $$ \sum_{i=1}^7 i  \quad \text{ and } \quad \sum_{j=4}^7 (j^2 +k) $$

> [!example] Problem
> Set up but **do not evaluate** a limit to approximate the area of the region between the function and the $x$-axis of:
> $$ f(x) := \frac{2x}{x^2+1} \quad \text{ on the interval } \quad [1,3] $$
>> [!success]- Solution
>> We can take a right hand limit, so let’s set up $R_n$ first:$$\Delta x = \frac{3-1}{n} = \frac2n \quad \text{ and } \quad x_i^* = 1 + \frac{2i}{n}$$
>> and just plug in:$$R_n = \sum_{i = 1}^n f\left ( 1 + \frac{2i}{n} \right )\cdot \frac{2}{n} = \sum_{i = 1}^n \frac{2 \left ( \frac{2i}{n} \right )}{\left ( \frac{2i}{n} \right )^2 + 1} \cdot \frac{2}{n}$$
>> To complete our answer, just add a limit in: $$A = \lim_{n \to \infty} R_n =  \lim_{n \to \infty} \sum_{i = 1}^n \frac{2 \left ( 1 + \frac{2i}{n} \right )}{\left ( 1 + \frac{2i}{n} \right )^2 + 1} \cdot \frac{2}{n} $$
